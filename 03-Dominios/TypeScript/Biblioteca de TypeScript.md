# Biblioteca de TypeScript

> TypeScript tem um equivalente direto claro — **Total TypeScript**, do Matt Pocock — formato hands-on baseado em exercícios no IDE com explicações em vídeo. A parte gratuita já é absurdamente boa.

## Equivalente direto

- **[Total TypeScript](https://www.totaltypescript.com)** (Matt Pocock) — portal de cursos + tutoriais gratuitos — O que mais se aproxima do papel do Real Python para TS. Tutoriais gratuitos (Beginner's, Zod) + workshops pagos para type-level programming avançado.
  - [Tutoriais gratuitos](https://www.totaltypescript.com/tutorials)

## Referência e fundamentos

- **[TypeScript Handbook](https://www.typescriptlang.org/docs/handbook/)** — documentação oficial — Surpreendentemente bem escrita para um handbook oficial.
- **[Effective TypeScript](https://effectivetypescript.com)** (Dan Vanderkam) — livro — Formato "62 specific ways to improve your TS". Para quem já passou do básico.

## Prática e desafios

- **[Type Challenges](https://github.com/type-challenges/type-challenges)** — repositório de exercícios — Para treinar type-level programming na marra.

## Aplicação prática

- **[tkdodo blog](https://tkdodo.eu/blog/)** (Dominik Dorfmeister, mantenedor do TanStack Query) — blog pessoal — Posts ótimos sobre TS aplicado a React/Query. Diretamente relevante para quem trabalha com React + TypeScript.
