# What can an RPA Bot actually do?

## Triggers

- Email

- Presence of a file or folder

- Scheduled bot run

- Object Interface trigger (click a button or keystroke)

- Specific window opening/closing

- Windows service changes state

- API call to the control room

## Acquire Data

- Extract data from an email.

- Retrieve a task from workflow system,

- Extract data from an excel spreadsheet

- Extract data from a webpage or thick client app.

- Read from database.

- Get data from API

## Use The Data

- Open local applications or websites.

- Fill out details of a web form.

- Compare data from multiple sources to validate it.

- Navigate a webpage

- Copy data from one application to another.

- Update a system of record.

## Delivery

- Send an email with results.

- Update/create an output file on a shared drive.

- Update a database with results from the bot run.

- Update a workflow application.

- Log a record of all actions taken and business rules  
    executed (audit of actions).
